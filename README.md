# web-components-exp

# falls serve noch nicht installiert (use serve anstelle von npx serve)

npm install

# Initialisieren der Projekte
npm run bootstrap

npm run build:clients

npm run serve:clients

## Starten der Shell Anwendung
npm run serve:shell

npm run build:serve

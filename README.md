# web-components-exp

# Installieren
npm install

# Initialisieren der Projekte 
npm run bootstrap

# Entwicklung
## Bauen der Clients
npm run build:clients

## Starten der Clients
npm run serve:clients

## Starten der Shell Anwendung
npm run serve:shell

# Installieren der Clients in Shell und Starten der Shell
npm run build:serve
